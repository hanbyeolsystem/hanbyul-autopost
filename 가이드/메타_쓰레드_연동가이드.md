# 인스타그램·페이스북·쓰레드 자동발행 — 설정 가이드

> 코드는 이미 배포됨. 아래 값들을 발급해 **Supabase 시크릿**에 넣으면 콘솔 "바로발행"이 자동발행으로 동작.
> Supabase 시크릿 위치: https://supabase.com/dashboard/project/jrzesjgyrvgvwazfajec/settings/functions

## 넣어야 할 시크릿
| 시크릿 이름 | 값 | 쓰는 곳 |
|---|---|---|
| `META_PAGE_ID` | 페이스북 페이지 ID | 페북·인스타 |
| `META_PAGE_TOKEN` | 장기 페이지 액세스 토큰 | 페북·인스타 |
| `META_IG_USER_ID` | 인스타 비즈니스 계정 ID | 인스타 |
| `THREADS_USER_ID` | 쓰레드 사용자 ID | 쓰레드 |
| `THREADS_TOKEN` | 쓰레드 액세스 토큰 | 쓰레드 |

> 참고: 본인 소유 페이지/계정에 올리는 것은 **앱 개발(Development) 모드**로 충분 — 정식 App Review 불필요.

## STEP 1. 인스타 비즈니스 전환 + 페이지 연결 (인스타 앱에서)
1. 인스타 앱 → 프로필 → ☰ → 설정 → **계정 유형 및 도구 → 프로페셔널(비즈니스)로 전환**
2. 같은 메뉴에서 **페이스북 페이지와 연결**

## STEP 2. Meta 개발자 앱 만들기
1. https://developers.facebook.com/ → 오른쪽 위 **내 앱 → 앱 만들기**
2. 유형 **비즈니스** 선택, 이름 아무거나(예: 한별광고)
3. 만든 앱 → 좌측 **앱 설정 → 기본**에서 `앱 ID`, `앱 시크릿` 메모

## STEP 3. 페이지 토큰·ID·인스타 ID 받기 (그래프 API 탐색기)
1. https://developers.facebook.com/tools/explorer/ 접속, 우측 상단에서 **내 앱 선택**
2. **권한 추가**(Add a Permission)에서 체크:
   `pages_show_list`, `pages_manage_posts`, `pages_read_engagement`, `instagram_basic`, `instagram_content_publish`, `business_management`
3. **Generate Access Token** → 페이스북 로그인·권한 동의
4. 요청창에 `me/accounts` 입력 → 실행 → 결과에서 내 페이지의
   - `id` → **META_PAGE_ID**
   - `access_token` → (단기) 페이지 토큰
5. **장기 토큰으로 교환** (브라우저 주소창에 입력, 값 채워서):
   `https://graph.facebook.com/v21.0/oauth/access_token?grant_type=fb_exchange_token&client_id=앱ID&client_secret=앱시크릿&fb_exchange_token=위에서받은토큰`
   → 나온 `access_token`(장기) → 다시 `me/accounts` 호출하면 **장기 페이지 토큰** = **META_PAGE_TOKEN**
6. 인스타 계정 ID: 탐색기에서 `페이지ID?fields=instagram_business_account` 실행
   → `instagram_business_account.id` = **META_IG_USER_ID**

## STEP 4. 시크릿 등록 → 페북·인스타 테스트
- Supabase 시크릿에 `META_PAGE_ID`, `META_PAGE_TOKEN`, `META_IG_USER_ID` 저장
- 콘솔에서 사진 1장 + 글 생성 → 페이스북/인스타 카드 **바로발행** 테스트

## STEP 5. 쓰레드 (별도)
1. 같은 개발자 앱에 **Threads API** 제품 추가 (또는 별도 Threads 앱)
2. 권한: `threads_basic`, `threads_content_publish`
3. Threads OAuth로 토큰 발급 후:
   - 사용자 ID: `https://graph.threads.net/v1.0/me?fields=id&access_token=토큰` → **THREADS_USER_ID**
   - 그 토큰 → **THREADS_TOKEN**
4. 시크릿 등록 → 쓰레드 카드 바로발행 테스트

---

## 동작 방식 (참고)
- 미디어는 한별 **Storage 공개 URL**로 전달 → Meta/Threads가 직접 가져감 (사진·영상 모두 사용)
- 인스타: 1장=단일, 여러 장=캐러셀, 영상=릴스(처리 대기 후 발행)
- 페북: 사진 여러 장=한 게시물(첨부), 영상=페이지 영상
- 토큰 미설정 시 콘솔은 글 복사 + 안내로 폴백(수동 발행 가능)
