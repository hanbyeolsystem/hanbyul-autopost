> 목적: 잘되는 경쟁사의 "이긴 공식"을 추출 → 한별시스템이 그대로 따라 만들 수 있게 정의

## ⭐ 모든 에이전트 공통 절대 원칙 (전 채널 적용)

1. **글과 사진은 반드시 일치한다.** 업로드한 사진을 Claude Vision으로 분석한 결과를 글에 반영하고, 사진에 없는 장비·모델·상황을 글에서 지어내지 않는다. (글과 그림이 어긋나면 신뢰가 무너짐 — 시스템에 강제 적용됨)
2. **고객의 어려움(증상)에서 출발한다.** 증상 → 공감 → 해결 → 안심. 파는 글이 아니라 돕는 글.
3. **핵심 키워드는 글 분석으로 자동 생성 가능.** 비워두면 AI가 지역·서비스·모델·증상을 바탕으로 검색에 유리한 키워드를 스스로 정해 녹인다.
4. **과거 발행 사례를 학습한다.** 채택(예약)한 글은 채널별 메모리에 저장되어, 다음 생성 시 톤·구성의 참고자료가 된다 → 쓸수록 일관되고 좋아진다.
5. **AI 그림은 보조 수단.** 실제 현장 사진이 최우선. 사진이 없을 때만 DALL·E 일러스트를 보조로 사용.
6. **마무리는 상호 + 연락처만.** 글 끝에 주소·"오시는 길"·찾아오는 길은 넣지 않는다. 고객이 사무실로 방문하는 업종이 아니므로, **상호(한별시스템)와 전화(053-588-7119)**만 표기한다. (모든 채널 공통)

## 2. 이긴 제목 공식 (조회수·노출 잘 나오는 패턴)

실데이터에서 반복적으로 발견된 제목 골격:

### A. 후기형 (가장 많음 / 클릭률 높음)
```
[지역] + [서비스/모델] + 설치 후기 / 솔직 후기 / 추천 후기
예) "부산 복합기 렌탈 업체 추천 임대 대여 솔직 추천후기!"
예) "평촌 학원 복합기렌탈 임대 후기 ㅣ 마이프린터 설치 후"
```
→ 핵심: **지역 키워드 + 업종 + "후기"** 3종 세트. 검색 유입의 왕도.

### B. 가이드형 (체류시간·신뢰 높음)
```
[대상] + [서비스] + 가이드 / 선택 기준 N가지 / 비교
예) "시놀로지 NAS 구축 업체 선택 기준 5가지"
예) "회사 파일 서버 구축, 중소기업 NAS 추천 – 규모별 서버 도입 가이드"
예) "NAS 서버 구축 가이드 및 시놀로지 vs 유그린 나스 비교"
```
→ 핵심: **숫자(5가지)·비교(vs)·규모별** 키워드가 클릭을 부른다.

### C. 사례형 (B2B 신뢰 최강)
```
[업종/규모] + [모델] + 구축/설치 사례
예) "시놀로지 NAS, 영어학원 사무실 DS925+ 구축/설치 사례"
예) "소규모 사무실 NAS 구축 추천! Synology DS224+ 설치사례"
```
→ 핵심: **"우리 같은 회사도 했구나"** 동질감. 잠재고객이 자기를 대입한다.

## 3. 이긴 본문 구조 (3대 템플릿)

### 템플릿 ① 후기형
1. 인사 + 어떤 고객/현장이었는지 (1줄)
2. 고객의 상황·고민 (공감 포인트) ★
3. 추천한 모델과 이유 (왜 이게 맞았나)
4. 설치 과정 (사진 2~3장)
5. 솔직 평가 + 만족 포인트
6. 이런 분께 추천 / 마무리
7. 회사 정보 + 해시태그

### 템플릿 ② 가이드형
1. 인사 + 누가 이 글을 봐야 하나
2. 왜 필요한가 (문제 정의)
3. 선택지 비교 (모델/브랜드/방식) — 표 활용
4. 상황별 추천 (규모별/용도별)
5. 선택 기준 N가지
6. 구축 시 주의점
7. 회사 정보 + 해시태그

### 템플릿 ③ 사례형 (B2B)
1. 인사 + 어떤 고객사였나 (업종·규모)
2. 고객사의 요청사항·문제
3. 제안한 구성 (네트워크 구조/모델)
4. 구축 과정 (현장 사진)
5. 도입 효과 (전후 비교)
6. 비슷한 고민이면 상담 (가벼운 도움 제안)
7. 회사 정보 + 해시태그

## 4. 공통 SEO·해시태그 규칙 (모든 경쟁사가 지킴)

- 제목 앞쪽에 **지역+핵심키워드** 배치 (예: "대구복합기렌탈")
- 본문에 핵심 키워드 3~5회 자연 반복
- 해시태그 10~15개: `#지역+키워드`, `#키워드`, `#업종`, `#모델명`, `#업체명` 조합
- 모델명(DS925+, GX7192, HP8710 등)을 제목·본문에 정확히 명기 → 모델 검색 유입

## 5. 한별시스템의 차별화 포인트 (가후의 전략)

경쟁사를 따라가되, **이기려면 한별만의 무기 3개를 얹는다:**

1. **"파는 곳"이 아니라 "돕는 곳" 톤** (이미 합의한 철학)
   → 경쟁사 대부분은 은근히 판매 지향. 한별은 "꼭 저희가 아니어도"로 차별화.

2. **시놀로지 공식 대리점 + AI 결합** (코아아이티가 "공식 파트너"로 먹는 권위를, 한별은 "공식 대리점 + AI 자동화 컨설팅"으로 한 단계 위에서)
   → "NAS를 단순 저장소가 아니라 AI가 일하는 데이터 거점으로."

3. **통합 관리(원스톱)** (신성CNS의 "한 업체에서" 메시지 강화)
   → 컴퓨터·프린터·NAS·예지정비까지 한 번에. "장비마다 업체 다른 피곤함" 해결.
